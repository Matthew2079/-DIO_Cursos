Criando um repositório

--> Inicializar o repositório com $ git init.
* Primeiro uso solicita:
** $ git config --global user.email "seu email"
** $ git congig --global user.name "seu usuario"

-> Adicionando um arquivo.

2. retornar ao Git Bash
3. dentro da pasta que está o repositorio ex: User@Matthew-Laptop MINGW64 /c/Workspace/livro-receitas (master)
